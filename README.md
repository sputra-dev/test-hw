## Feature Application

- Register as member
- Login admin or member
- Create, Update, Delete Categories
- Create, Update, Delete Books
- Loan Books
- Restore Books
- Get list loan books

## Run Pre-Test

- http://127.0.0.1:8000/looping

## Installation

---
cp .env.example .env
---
---
php artisan key:generate
---
---
composer install
---
---
php artisan migrate --seed
---
---
php artisan serve
---
